# Title: M-02 Partition Polygon Clipping + 2.5D Style
Description: 파티션 폴리곤 경계로 이미지를 잘라 매핑하는 방식. 경계선이 너덜너덜해서 대부분 실패. 특수 조건에서만 허용.
When-To-Read: 파티션 개별 이미지 클리핑 방식을 고려할 때
Keywords: partial, polygon, clipping, partition, boundary, 2.5d
Priority: low

**날짜**: 2026-04-21

---

## 결과 요약

- 2.5D 스타일 자체는 방향 맞음
- 건물/오브젝트가 너무 많이 섞여 나옴
- **폴리곤 클리핑 경계선이 너덜너덜** → 시각적으로 허용 불가

---

## 폴리곤 클리핑 포기 결정

파티션 형태가 복잡할수록 클리핑 경계가 심해진다. feather/blur로 완화해도 경계가 눈에 보인다. 인접 파티션 이어 붙이면 단절이 더 심해진다.

**대안**: tile 모드 (정사각형 출력) + 연속 지형 시스템 UV 타일링 → [M-01] 참고

---

## 허용 조건 (예외적으로 사용 가능한 경우)

- 특수 이벤트 지역의 "고의적 경계" 연출이 필요한 경우
- 파티션 경계가 단순한 직사각형에 가까운 경우
- 경계선 자체를 게임 연출로 활용하는 경우 (예: 마법진 경계)
