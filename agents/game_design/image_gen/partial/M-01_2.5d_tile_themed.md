# Title: M-01 2.5D Tile Themed Textures
Description: 2.5D topview 스타일 테마 타일 텍스처 (얼음/도심/해변). 게임 아트 느낌은 있지만 오브젝트 간섭 있음. 조건부 사용.
When-To-Read: 2.5D 게임 아트 스타일 타일 텍스처 팔레트가 필요할 때
Keywords: partial, 2.5d, tile, lora, zavy, detail, ice, urban, beach, themed
Priority: medium

**날짜**: 2026-04-21  
**스크립트**: `back/scripts/generate_partition_textures.py --tile`  
**출력 경로**: `front/public/ground/25d_test/`

---

## 설정값

| 항목 | 값 |
|------|-----|
| 모델 | `juggernautXL_v10.safetensors` |
| LoRA 1 | `zavy-ctsmtrc-sdxl.safetensors` strength=0.7 |
| LoRA 2 | `add-detail-xl.safetensors` strength=0.6 |
| steps | 30 |
| CFG | 6.0 |
| tile-px | 768 → 출력 **1536px** |
| 폴리곤 클리핑 | **없음** (tile 모드 필수) |

## 주의사항

- 오브젝트(소품, 건물 잔재)가 간혹 섞여 나옴 → negative 강화 필요시 추가
- seamless 타일링 시 패턴 반복이 눈에 띌 수 있음
- tile 모드는 정사각형 출력 → 파티션 1:1 매핑 불가
- 정글 계열은 절대 이 방식으로 생성 금지 → [F-01] 참고

---

## 사용 조건

- 특정 테마의 타일 텍스처 팔레트로 사용할 때 적합
- 파티션 1:1 매핑보다 연속 지형 시스템의 재질 소스로 쓸 것

---

## 실행 명령

```bash
python back/scripts/generate_partition_textures.py \
  --partition-keys seoul..2.v2.0038 \
  --style ice_rpg \        # 또는 urban_rpg, beach_rpg
  --lora zavy-ctsmtrc-sdxl.safetensors --lora-strength 0.7 \
  --lora2 add-detail-xl.safetensors --lora2-strength 0.6 \
  --steps 30 \
  --tile --tile-px 768 \
  --output-dir front/public/ground/25d_test
```
