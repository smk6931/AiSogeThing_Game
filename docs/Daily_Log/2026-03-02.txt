=== 2026-03-02 데일리 로그 ===

오전 (업무 보고)
1. 월드 오브젝트 배치 효율화를 위한 GUI 툴 도입 (lil-gui)
2. 실시간 환경 파라미터(안개, 조명) 튜닝 시스템 구축
3. 건물 등 에셋의 좌표/회전/스케일 실시간 수정 기능 구현

진행 사항
1. front 폴더에 lil-gui 라이브러리 설치 완료
2. WorldDebugger.jsx 컴포넌트 신설
   - 환경(Environment) 폴더: 안개 거리, 색상, 조명 강도 조절
   - 오브젝트 에디터(Object Editor): 이동/회전/크기 모드 전환 피드백
   - 저장 유틸리티: 현재 설정값을 콘솔에 출력하는 기능 포함
3. RpgWorld.jsx 연동
   - TransformControls와 lil-gui를 연동하여 기즈모 모드 실시간 변경 가능
   - 상태값(debugConfig)을 통해 변경 사항 즉시 반영

남은 과제
1. 실제 맵 설정 파일(mapConfig.js)로의 자동 저장 API 연동 검토
2. 다른 주요 오브젝트들도 선택하여 수정 가능하도록 확장
