# 스킬 시스템 고도화 구현

## 배경

`docs/plan/2026-04-12/2100_스킬_시스템_고도화.md` 계획에 따라 Phase 1-3 전체 구현.

## 구현 범위

### Phase 1 — 스킬 카탈로그 + 쿨다운 기반

**신규**
- `front/src/data/skillCatalog.js`: 4개 스킬 전체 정의 (icon, mpCost, cooldownMs, type, damage, mult, range)
- `front/src/hooks/useSkillHotbar.js`: 슬롯 배열, 쿨다운 추적(RAF), MP 소모, localStorage 동기화

**수정**
- `back/player/services/service.py`: `SKILL_POWER` → `SKILL_CATALOG` (multiplier 반영) + `handle_frost_nova()` 추가
- `front/src/entity/player/logic/usePlayerSkills.js`: skill1(Q)~skill4(F) 4키 엣지 감지 → `onTrigger(slotIdx, pos, rot)` 호출
- `front/src/engine/useGameInput.js`: q/e/f 키 추가, actions.skill1~4 매핑

### Phase 2 — 새 스킬 투사체

**신규**
- `front/src/entity/player/projectile/LightningBoltProjectile.jsx`: 노란 실린더, speed 30, 2개 관통
- `front/src/entity/player/projectile/FrostNovaEffect.jsx`: 빙결 링 애니메이션 0.55s, AoE 피격 sendHit

**수정**
- `back/monster/managers/MonsterManager.py`: `batch_hit()` 메서드 추가 (AoE용)
- `back/player/routers/router.py`: frost_nova 분기 → `handle_frost_nova()` 호출 + `frost_nova_hits` 브로드캐스트
- `front/src/engine/useGameSocket.js`: `frost_nova_hits` 케이스 추가, skill broadcast에 frost_nova 분기

### Phase 3 — 퀵슬롯 UI + 드래그앤드랍

**신규**
- `front/src/ui/SkillHotbar.jsx`: 4슬롯 바, conic-gradient 쿨다운 sweep, MP 부족 표시, onDrop
- `front/src/ui/SkillPanel.jsx`: 스킬 목록 + HTML5 draggable, 레벨 잠금 표시

**수정**
- `front/src/ui/GameOverlay.jsx`: SkillHotbar + SkillPanel lazy import, showSkillPanel 상태, 토글 버튼
- `front/src/engine/GameCanvas.jsx`: `skillHotbar` prop 추가 전달
- `front/src/GameEntry.jsx`: `useSkillHotbar` 인스턴스화, `localMp` 상태, GameCanvas/GameOverlay 양쪽 전달
- `front/src/entity/world/RpgWorld.jsx`: LightningBoltProjectile/FrostNovaEffect import, `skillHotbar` prop으로 슬롯 발동, onAction → slot 기반 분기
- `front/vite.config.js`: `@data` alias 추가

## 아키텍처 결정

- `useSkillHotbar`는 GameEntry에서 단일 인스턴스화 → GameCanvas(발동용) + GameOverlay(UI용) 양쪽 전달
- frost_nova AoE 계산은 서버에서만 처리 (클라이언트는 `sendHit({ skillName: 'frost_nova' })` 만 전송)
- 쿨다운 추적은 RAF 기반 (`cooldownEndsRef` + `requestAnimationFrame`) — state 업데이트 최소화

## 검증 방법

1. Q/E/R/F 키로 각각 스킬 발동 확인
2. 쿨다운 sweep 애니메이션 확인
3. ⚡ lightning_bolt: 직선 발사, 2개 몬스터 관통
4. ❄️ frost_nova: 빙결 링 이펙트, 주변 몬스터 피격
5. 스킬 패널 열고 드래그로 퀵슬롯 교체
6. 브라우저 새로고침 후 슬롯 배치 유지 (localStorage)
