# 2026-03-27 Yongsan World Table Draft

## 목적

용산구를 첫 번째 고도화 구역으로 삼기 위해, 실제 행정동 단위 위에 월드 성격 데이터를 얹는 초안을 만든다.

이 문서는 아래 두 가지를 동시에 만족시키기 위한 기준 문서다.

- 월드 디자이너 관점: 동마다 다른 감정, 분위기, 플레이 이유를 정의
- 시스템 관점: 바닥 텍스처, 몬스터, 자원, 이벤트를 데이터 기반으로 연결

실제 데이터 파일은 아래에 저장했다.

- [yongsan_world_profile.json](C:/GitHub/AiSogeThing_Game/back/world/data/yongsan_world_profile.json)

## 왜 지금 테이블부터 필요한가

텍스처를 먼저 고르면 나중에 뒤집을 일이 많다.

먼저 필요한 것은:

1. 이 동이 어떤 구역인가
2. 낮/밤 분위기가 어떤가
3. 여기는 사람들이 왜 모이나
4. 무슨 자원이 나오나
5. 어떤 몬스터가 어울리나

이걸 먼저 정해야 텍스처가 "예쁜 그림"이 아니라 "기능하는 월드"가 된다.

## 지금 초안의 성격

이건 완성본이 아니라 운영 가능한 형태를 염두에 둔 `1차 월드 메타데이터`다.

즉 지금 단계 목표는:

- 용산구 각 동의 성격을 강제로라도 구분한다
- 동마다 텍스처 선택 기준을 만든다
- 몬스터/자원/이벤트를 붙일 자리를 만든다

## 내가 보는 가장 중요한 포인트

용산구의 장점은 한 구 안에서 다음이 모두 가능하다는 점이다.

- 한강 수변 생활
- 이태원형 야간 소셜
- 한남동형 세련된 상류 문화
- 후암/보광형 밀도 높은 골목 탐험
- 한강로/원효로형 현실 도시 인프라

즉 용산은 "서울의 축소판"처럼 쓰기 좋다.

그래서 첫 고도화 구역으로 적합하다.
