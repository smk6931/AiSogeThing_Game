# Codex Workspace Notes

This folder is a project-local guidance layer for this repository.

It is not an official Codex runtime feature like home-installed skills.
It exists to keep project rules, implementation notes, and feature workflows split into smaller files instead of growing `AGENTS.md` forever.

Use it for:

- feature-specific implementation rules
- UI direction notes
- auth flow notes
- player progression notes
- world content notes

Recommended rule:

- keep `AGENTS.md` small and global
- keep evolving project decisions in `codex/`
- move stable reusable guidance into a real skill only when it becomes cross-project or tool-specific
